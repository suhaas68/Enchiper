# HTML Semantic Tasks – Output Documentation

This repository contains multiple HTML tasks based on semantic tags, multimedia embedding, forms, navigation, and structured webpage layout.

## **Task 1 – Full Blog Homepage Using Semantic Tags**
### **Description**
A complete homepage using semantic tags such as:
`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`.

### **Output Contains**
- A main blog title and subtitle  
- Navigation bar  
- Two latest article previews  
- Sidebar with popular topics  
- Footer with copyright  

---

## **Task 2 – Video Gallery Page**
### **Description**
A simple video gallery using `<video>` inside semantic `section` and `article`.

### **Output Contains**
- Two articles  
- Each article includes a heading + playable video  
- Uses only semantic tags, no CSS  

---

## **Task 3 – Music Player Page**
### **Description**
A webpage that embeds MP3 files using `<audio controls>` inside articles.

### **Output Contains**
- A title section  
- Two audio players in separate articles  
- Footer with a message  

---

## **Task 4 – Education Website Homepage**
### **Description**
Homepage for an educational institute using multimedia embeds.

### **Output Contains**

- Welcome video (`<video>`)  
- YouTube embedded Campus Tour (`<iframe>`)  

---

## **Task 5 – News Portal Using Semantic Tags**
